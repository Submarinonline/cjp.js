# cjp (correctJP)

[![npm version](https://badge.fury.io/js/cjp.svg)](https://badge.fury.io/js/cjp)

怪レい翻訳ライブラリ

## 使い方(Node)

```js
const { generate } = require('cjp')

console.log(generate('おはようございます'))
```

## 使い方

`examples/`に何個かサンプルを置いてます。

#### Node, Vue

サンプルがそのまま使えます

#### HTML 手書きの場合

`examples/example.browser.js`を parcel 等でバンドルしましょう。
